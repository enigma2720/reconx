# 🔍 ReconX - Advanced Bash Recon Toolkit

## Overview

ReconX is a modular Bash-based reconnaissance toolkit built for ethical hackers and penetration testers.

It automates:
Recon → Probe → Scan → Analyze → Report

---

## Features

* Subdomain Enumeration (subfinder, assetfinder)
* Live Domain Detection (httpx)
* Nmap Port Scanning
* Directory Bruteforce (ffuf)
* Tech Detection (whatweb)
* DNS & WHOIS Lookup
* Auto Tool Installation

---

## Installation

```bash
git clone https://github.com/enigma2720/reconx.git
cd reconx
chmod +x reconx.sh
```

---

## Usage

```bash
./reconx.sh -d example.com
```

---

## Output

All results are saved in structured folders under `/output/`.
